# Аналіз та прогнозування звільнення працівників

## Мета проєкту
Аналіз ключових факторів, що впливають на утримання працівників у компанії, та розробка моделі машинного навчання для прогнозування того, чи залишить співробітник компанію.

## Джерело даних
https://www.kaggle.com/datasets/tawfikelmetwally/employee-dataset

## Гіпотези для аналізу
1. **Фінансова мотивація:** Співробітники з нижчим рівнем оплати (Payment Tier) частіше схильні до звільнення, ніж ті, хто знаходиться у вищому зарплатному тирі.
2. **Залученість у проєкти:** Працівники, які тимчасово залишалися без роботи ("Ever Benched" = Yes), мають вищу ймовірність покинути компанію через втрату мотивації.
3. **Стаж роботи:** Існує критичний період лояльності - співробітники, які приєдналися до компанії нещодавно (Joining Year), звільняються значно частіше, ніж ті, хто працює понад 3-4 роки.
