# Changelog

## [v0.1.0] - 2026
### Додано
- Ініціалізація проєкту та створення структури папок.
- Налаштування `.gitignore` для ігнорування віртуального середовища та сирих даних.
- Скрипт `src/data_load.py` для завантаження датасету Employee.
- Скрипт `src/data_quality.py` для перевірки даних на пропуски та дублікати.
- Скрипт `src/data_research.py` для аналітичної перевірки гіпотез та ML-модель Random Forest для прогнозування звільнень.
- Скрипт `src/visualization.py` для генерації графіків та їх збереження у папку `reports/figures/`.
- Вирішено конфлікт злиття (merge conflict) у файлі `README.md`.
